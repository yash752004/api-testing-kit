# Web App

This is the SvelteKit frontend for the API Testing Kit monorepo.

## Run

From the repo root:

```bash
bun run dev:web
```

Or directly inside this app:

```bash
bun run dev
```

## Check

```bash
bun run check
```

## Test

```bash
bun run test
```

## Notes

- App path: `apps/web`
- Backend path: `server`
- Product and implementation docs live under `doc/`
